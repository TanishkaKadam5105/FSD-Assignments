# Assignment 2 – Personal Portfolio Website

## Description

This assignment is a **personal portfolio website** created using HTML, CSS, and JavaScript.
The portfolio presents information about the student, including skills, projects, internships, and contact details in a clean and responsive layout.

The website demonstrates basic **front-end development concepts and UI design**.

## Features

* Responsive portfolio layout
* Navigation bar for easy section access
* About section with skills list
* Projects section with clickable project details
* Experience and internship information
* Contact section with social links
* Interactive project detail view using JavaScript
* Modern UI design using CSS styling

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Inter)

## Sections in the Portfolio

### 1. Header

Displays the student name, role, and navigation menu.

### 2. About

Contains a short introduction and a list of technical skills such as:

* C
* C++
* Java
* Kotlin
* Python
* HTML, CSS, JavaScript
* PHP & MySQL
* Figma

### 3. Projects

Shows major projects including:

* Front-End Project (Diploma Internship)
* Android App Development Project
* Companion Circle Community Engagement Project

Each project can be clicked to view detailed information.

### 4. Experience

Lists internship experience and practical development work.

### 5. Contact

Provides ways to connect with the student including:

* LinkedIn
* GitHub
* Blog

## Purpose of the Assignment

The objective of this assignment is to:

* Practice creating a **personal portfolio website**
* Implement **responsive web design**
* Organize project information clearly
* Use **basic JavaScript for interactivity**
