# Assignment 1 – Student Table and HTML Tags Table

## Description
This assignment demonstrates the use of **basic HTML structure and table elements**.  
The webpage contains two main sections:
1. A **Student Table** displaying student IDs and names.
2. An **HTML Tags Table** showing commonly used HTML tags, their syntax, and their purpose.

CSS styling is applied to improve the appearance of the tables and webpage.

## Features
- Display of student data in a table format
- HTML tags reference table
- Basic CSS styling for layout and design
- Hover effect on table rows
- Centered layout using div and flexbox

## Technologies Used
- HTML5
- CSS3

## Tables Included

### 1. Student Table
Displays a list of students with:
- Student ID
- Student Name

### 2. HTML Tags Table
Shows commonly used HTML tags with:
- Tag name
- Syntax
- Use/Purpose

Examples of tags included:
- `<!DOCTYPE>`
- `<html>`
- `<head>`
- `<title>`
- `<table>`
- `<tr>`
- `<th>`
- `<td>`
- `<div>`
- `<img>`
- `<script>`
- `<span>`
- `<style>`

## Purpose of the Assignment
The goal of this assignment is to understand:
- Basic HTML page structure
- Table creation using HTML
- Styling elements using CSS
- Organizing content in a structured format
