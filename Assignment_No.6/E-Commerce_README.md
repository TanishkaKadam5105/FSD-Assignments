# Assignment 3 – E-Commerce Website (FashionHub)

## Description

This assignment is a **basic E-Commerce website** named **FashionHub** developed using HTML, CSS, and JavaScript.
The website allows users to browse products, filter categories, add items to the cart, and proceed to checkout.

The project demonstrates the basic structure and functionality of an online shopping platform.

## Features

* Product listing page
* Search functionality for products
* Category filters (Men / Women / All)
* Add products to cart
* Cart page to view selected products
* Total price calculation
* Checkout page with payment form
* Responsive design using CSS

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Website Pages

### 1. Home Page (index.html)

* Displays available products in a grid layout
* Provides search functionality
* Category filter buttons (Men / Women / All)
* Navigation bar with cart link

### 2. Cart Page (cart.html)

* Shows all products added to the cart
* Displays total price of selected items
* Option to proceed to checkout

### 3. Checkout Page (checkout.html)

* Form for entering user details such as:

  * Name
  * Address
  * Card Number
* Button to complete payment

### 4. Styling (style.css)

* Custom styling for:

  * Navigation bar
  * Product cards
  * Buttons
  * Cart layout
  * Checkout form
* Responsive layout for mobile and tablet devices

## Purpose of the Assignment

The objective of this assignment is to:

* Understand the structure of an **E-Commerce website**
* Implement **product listing and cart functionality**
* Practice **HTML layout, CSS styling, and JavaScript interaction**
* Build a simple **shopping flow from product selection to checkout**
