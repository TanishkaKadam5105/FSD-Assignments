# Assignment 6 – Online Appointment Booking / Used Items E-Commerce Portal

## Description

This assignment is a web-based application developed using **Node.js**, **Express.js**, and **MongoDB**.

The project is based on the concept of an **Online Appointment Booking Application** or an **E-Commerce Portal for Used Item Sales** such as cars, bikes, electronics, furniture, and similar reusable products.

The website allows users to browse products or services, interact with the system, submit feedback, and manage data through a connected backend database.

The project demonstrates concepts of **full-stack web development**, **routing**, **server-side programming**, **database connectivity**, and **dynamic user interaction**.

## Features

* Responsive website layout
* Navigation bar for easy page access
* Product / service listing section
* Appointment booking or inquiry system
* Feedback page connected with MongoDB
* Data storage using NoSQL database
* Backend developed using Express.js
* User-friendly interface
* Localhost deployment support
