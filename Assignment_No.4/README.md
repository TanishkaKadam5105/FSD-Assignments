#Student Analytics Dashboard

A modern, responsive, and interactive analytics dashboard built using **HTML, CSS, and Chart.js**. This project visualizes student performance data through clean UI components and dynamic charts.

---

##Features

* Interactive charts using Chart.js
* Multiple dashboard pages (Overview, Analytics, Students, Settings)
* Clean and professional UI design
* Responsive layout for different screen sizes
* Student data table view
* Settings form interface
* Navigation with sidebar and back button

---

## Pages Included

* **Overview** – Summary cards, charts, and progress bars
* **Analytics** – Performance and attendance charts
* **Students** – Table displaying student records
* **Settings** – User input form

---

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript
* Chart.js
